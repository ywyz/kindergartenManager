# 🚀 快速开始 - 5分钟上手

欢迎使用幼儿园教案管理系统！本指南将帮助您在5分钟内快速上手。

---

## 📦 第一步：安装与启动

### 1.1 安装依赖
```bash
# 如果还没有安装过
conda env create -f environment.yml
conda activate teacher
```

### 1.2 启动应用
```bash
python app.py
```

### 1.3 打开浏览器
访问: **http://localhost:8080**

---

## ⚙️ 第二步：初始配置（1分钟）

1. 点击页面顶部的 **⚙️ 系统配置** 按钮
2. 进入 **AI配置** 标签页
3. 输入你的 OpenAI API Key (获取方式: [OpenAI官网](https://platform.openai.com/))
4. 点击 **保存配置**

### 配置项说明

| 项目 | 说明 | 是否必填 |
|------|------|--------|
| API Key | OpenAI API密钥 | ✅ 必填（使用AI时） |
| AI模型 | 默认 `gpt-4o-mini` | ⚪ 可选 |
| API地址 | 留空使用官方API | ⚪ 可选 |

> **提示**: 配置自动保存到浏览器，下次访问自动恢复

---

## 📝 第三步：创建教案（2分钟）

### 3.1 设置学期信息
1. 在 "学期开始日期" 输入框输入如: `2026-02-23`
2. 在 "学期结束日期" 输入框输入如: `2026-07-10`
3. 点击 **保存学期**

### 3.2 填写教案内容
1. 在 "教案日期" 选择一个日期（系统自动计算周次和星期几）
2. 逐个填写各个字段：
   - 晨间活动
   - 晨间活动指导
   - 晨间谈话
   - 集体活动
   - 室内区域游戏
   - 下午户外游戏
   - 一日活动反思

### 3.3 （可选）使用AI辅助
如果已配置API Key：
1. 在 "集体活动管理原稿" 输入框输入完整的活动描述
2. 点击 **分割集体活动** 按钮
3. AI 会自动填充：活动主题、活动目标、活动准备等6个子字段

---

## 💾 第四步：保存与导出（1分钟）

### 4.1 保存到数据库
- 点击 **保存到数据库** 按钮
- 教案数据将被存储，可随时加载

### 4.2 导出为Word
- 点击 **导出为Word** 按钮
- 生成的Word文件保存在 `output/` 文件夹中
- 可直接打印或继续编辑

---

## 🎯 常用操作速览

| 操作 | 按钮 | 说明 |
|------|------|------|
| 快速填充示例 | 填充测试数据 | 用示例数据快速测试系统 |
| 清空所有内容 | 清空表单 | 重新开始编写教案 |
| 加载已保存教案 | 加载到表单 | 从数据库恢复之前的教案 |
| 批量导出 | 连续导出 | 一次生成多个日期的文档 |
